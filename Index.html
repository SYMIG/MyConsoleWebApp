Open Console

Microsoft Visual Studio is an integrated development environment (IDE) released by Microsoft. It is used to create computer programs, web pages, web apps, web services and mobile apps.

To setup Visual Studio, first, download Visual Studio on your device. Next, complete the steps that appear when the download is complete. Then, create a folder for your project. Finally, open that folder in the Visual Studio and add 3 files. Name them index.html, main.js, and style.css respectively. Now you can start working on your project.

Windows Console is the infrastructure for console applications in Microsoft Windows. An instance of a Windows Console has a screen buffer and an input buffer. It allows console apps to run inside a window. To open the console in Microsoft Edge, you can use the shortcut: Control + Shift + I. As an alternative, you can right-click on the webpage and click "Inspect" to open the developer console. In the window that opens, click the "Console" tab. In the JavaScript file, use console.log() method to print on console.
